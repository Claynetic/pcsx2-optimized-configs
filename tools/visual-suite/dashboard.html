Quick Setup

First time? Run the setup script:

.\tools\setup.ps1 -PCSX2Path "C:\path\to\PCSX2"

This auto-detects your games and generates optimized configs.

ReShade Installation:

1. Download from https://reshade.me

2. Target: pcsx2-qt.exe with Direct3D 10/11/12

3. Install packages: Standard, SweetFX, AstrayFX, qUINT

4. Set AutoSavePreset=0 in ReShade.ini

Critical ReShade Config:

Add subdirectories to EffectSearchPaths:

Shaders\SweetFX, Shaders\AstrayFX, Shaders\qUINT

ReShade does NOT search recursively!

Controls & Commands

In-Game Controls:

Home — Open/close ReShade overlay

Scroll Lock — Toggle all effects on/off

PowerShell Commands:

.\switch_preset.ps1 Cinematic — Switch preset

.\switch_preset.ps1 -List — Show all presets

.\toggle_enhancements.ps1 — Enable/disable ReShade

Renderer Note: AMD RDNA2/3 GPUs have known Vulkan timeout issues in PCSX2 (#10720). Use DX11 (Renderer=3) for stability.
